# Automated Backup and Restore System

## Overview

A Python-based automated backup and restore system designed to protect user files from accidental deletion or system failure.

## Features

- Automated file backup
- Restore functionality
- Timestamp-based backup versions
- Backup logging
- File count monitoring
- Backup size monitoring
- ZIP compression support
- Backup history tracking

## Technologies Used

- Python
- OS Module
- Shutil Module
- Zipfile Module

## How to Run

python main.py
